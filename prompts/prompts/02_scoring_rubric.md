# Prompt 02 — Scoring Rubric + Recommendation

**Goal:** Score a job against Beth’s target profile using a transparent 100-point rubric, then output a clear recommendation.

## Prompt (copy/paste into ChatGPT)

You are an AI Job Search Agent. Using the structured job intake below, score this role for Beth Silverberg.

### Candidate positioning (use as context)
Beth is an AI adoption + GTM partnerships leader (not a software engineer). Strengths include partnerships, enterprise sales, sponsorship strategy, revenue operations, customer success, and GenAI workflow adoption. Prioritize remote, business-side AI roles.

### Scoring rubric (100 points)
Score each category and explain the reasoning briefly.

1) **Role alignment (25):** How well the title + responsibilities match target roles (AI partnerships, GTM, adoption, CS, program/project ops).  
2) **Transferable experience (20):** How directly Beth’s background maps to requirements (enterprise GTM, partnerships, client success, operations).  
3) **AI readiness match (15):** Does the role require GenAI adoption/enablement skills that match Beth’s training and positioning (prompting, workflows, change mgmt)?  
4) **Seniority match (10):** Is the scope appropriate for Beth’s seniority (manager/director-level business roles)?  
5) **Industry match (10):** AI startups/SaaS/infrastructure, healthcare AI, animal health tech, fintech AI, martech AI.  
6) **Remote + compensation practicality (10):** Remote-first fit, travel reasonable, comp range realistic if listed.  
7) **ATS keyword alignment (10):** How well Beth can mirror the role’s language and keywords.

### Automatic penalty (subtract 10–20 points if central to the role)
Apply and explain a penalty if the job centrally requires any of:
- software engineering / building production code
- ML model development / research
- technical product management with roadmap ownership for AI products
- SQL/Python/analytics stack as core requirements
- mandatory prior experience at an AI startup

### Final recommendation
Use these bands:
- **85–100 = APPLY NOW**
- **70–84 = STRONG APPLY**
- **55–69 = STRETCH APPLY**
- **<55 = SKIP**

### Output format (required)
**A) Score breakdown (table or bullets)**
- Role alignment: __/25
- Transferable experience: __/20
- AI readiness: __/15
- Seniority: __/10
- Industry: __/10
- Remote/comp practicality: __/10
- ATS alignment: __/10
- Penalty: -__/20
- **Total: __/100**

**B) Recommendation:** Apply Now / Strong Apply / Stretch Apply / Skip  
**C) Why (top 5 reasons)**  
**D) Risks / gaps (top 3)**  
**E) Best next step:** Apply Now / Network First / Tailor Resume First
