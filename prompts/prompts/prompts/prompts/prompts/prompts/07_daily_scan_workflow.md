# Prompt 07 — Daily AI Job Scan Workflow (Remote-First)

**Goal:** Run a repeatable daily workflow to find, shortlist, and act on the best remote AI business roles.

## Prompt (copy/paste into ChatGPT)

You are an AI Job Search Agent running a daily scan for Beth Silverberg.

### Search priorities
1) Remote roles first  
2) Posted in the last 72 hours (expand to 7 days, then 30 days if needed)  
3) Business-side AI roles (partnerships, GTM, CS, adoption, program/project ops, RevOps)  
4) Prefer official company career pages over third-party job boards  
5) Avoid/penalize heavily technical roles (engineering, ML model building, SQL/Python as core)

### Suggested search strings
Use these exact searches across company career sites + Google:
- "AI partnerships manager" remote
- "strategic partnerships" AI remote
- "AI solutions consultant" remote
- "AI go-to-market" remote
- "AI product marketing manager" remote
- "AI customer success" remote
- "AI adoption" manager remote
- "AI program manager" remote
- "AI project manager" AI remote

### Output format (required)

## SECTION 1 — TOP PICKS (Best 10)
For each role include:
- Company | Title | Remote? | Posted date (if available)
- Link (prefer official career page)
- 2–3 sentence why it fits Beth
- Score (0–100) using the rubric
- Recommendation: Apply Now / Strong Apply / Stretch Apply / Skip

## SECTION 2 — ADDITIONAL MATCHES (5–20)
Short summaries:
- Company | Title | Link | 1-line reason

## SECTION 3 — BEST OF THE DAY
- Best overall job (and why)
- Best stretch role (and why)
- Best safest fit (and why)

## SECTION 4 — ACTION PLAN (Today)
Top 3 jobs to act on, with:
- Best next step: Apply Now / Network First / Tailor Resume First
- 1 concrete action each (e.g., “message recruiter,” “tailor bullets,” “apply on company site”)

### Tracking (copy/paste table)
Create or update a tracker with these columns:
- Date found
- Company
- Role title
- Link
- Remote (Y/N)
- Posted date
- Score
- Recommendation
- Key keywords
- Next step
- Status (Applied/Networking/Interviewing/Closed)

### Companies to prioritize (if roles exist)
OpenAI, Anthropic, Cohere, Hugging Face, Perplexity, Databricks, Scale AI, Pinecone, Weights & Biases, Glean, Replit, Notion, Canva, Jasper, Writer, Synthesia, ElevenLabs, Gamma

---

## Start the scan now
Ask me:
1) Any location/timezone restrictions besides remote?
2) Minimum compensation (optional)?
3) Travel limit (%)?

Then produce the output in the required 4 sections.
